# Norway Travel

A travel guide and planning application for exploring Norway.

## Features

- Discover popular destinations
- Plan your itinerary
- Find local recommendations
- Track travel expenses

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm or yarn

### Installation

```bash
git clone https://github.com/kkonstantin42/v0-norway-travel.git
cd v0-norway-travel
npm install
```

### Running the Application

```bash
npm start
```

The application will open at `http://localhost:3000`
